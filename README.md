
# ⚙️ sysadmin-scripts

Colección de scripts para la administración, automatización y mantenimiento de servidores Linux. Aquí encontrarás herramientas que uso en el día a día para facilitar tareas repetitivas o complejas.

---

## 📁 Estructura

```
sysadmin-scripts/
├── backups/        # Scripts de copias de seguridad
├── monitoring/     # Monitorización de recursos y servicios
├── users/          # Gestión de usuarios y permisos
├── networking/     # Configuración y diagnóstico de red
├── maintenance/    # Actualizaciones, limpieza y mantenimiento
└── misc/           # Scripts varios
```

---

## 🚀 Uso

Cada script incluye comentarios explicando qué hace y cómo usarlo. Revisa siempre el contenido antes de ejecutarlo en producción.

```bash
# Dar permisos de ejecución
chmod +x script.sh

# Ejecutar
./script.sh
```

---

## 📋 Requisitos

- Linux (Debian/Ubuntu o derivados salvo que se indique lo contrario)
- Bash 4.0+
- Permisos de root o sudo según el script

---
